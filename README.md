# Blinkit-Sales-Analysis


## 📊 Blinkit Sales Analysis

This project focuses on analyzing Blinkit's sales data to extract meaningful business insights related to sales performance, customer satisfaction, and inventory distribution. The analysis is performed using key performance indicators (KPIs) and interactive dashboards to support data-driven decision-making.

The main objective of this project is to understand how different factors such as item type, fat content, outlet size, location, and establishment year impact overall sales and customer ratings.

### 🎯 Problem Statement

To conduct a comprehensive analysis of Blinkit’s sales data in order to evaluate performance, identify trends, and uncover insights using KPIs and visualizations.

### 📌 Key KPIs

* Total Sales
* Average Sales
* Number of Items Sold
* Average Customer Rating

### 📈 Analysis Performed

* Sales analysis based on item fat content
* Performance comparison across different item types
* Outlet-wise sales analysis by size, location, and type
* Impact of outlet establishment year on sales
* Sales distribution across Tier 1, Tier 2, and Tier 3 locations

### 🛠️ Tools & Technologies

* Python (Data Cleaning & Processing)
* Excel (Data Handling)
* SQL (Data Analysis)
* Power BI (Dashboard & Visualization)

### 📊 Features

* Interactive dashboards with KPI tracking
* Visual analysis of sales distribution
* Outlet and product performance insights
* Data-driven business recommendations

### 🚀 Conclusion

This project demonstrates how raw retail data can be transformed into actionable insights using data analytics tools,
helping businesses improve sales strategies and customer satisfaction.

### 🔮 Future Scope

* Sales forecasting using Machine Learning
* Real-time dashboard integration
* AI-based recommendation systems
